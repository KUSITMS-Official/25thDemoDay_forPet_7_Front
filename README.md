## 프로젝트 소개
온/오프라인 지도 서비스와 반려가구 커넥트를 형성하여 완벽한 반려생활의 나침반이 되고자 기획한 반려견 원스톱 플랫폼

### 개발 언어 / 사용 라이브러리
Typescript, React
